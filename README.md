# Tic-Tac-Toe Game

## Overview
A **C#**-based **Tic-Tac-Toe** game with a **graphical user interface (GUI)**, created as the final project for the **Object-Oriented Programming** course.

## Features
- Two-player gameplay
- Basic game logic
- Graphical interface

## Installation
1. Clone the repository: `git clone <repository-link>`
2. Open in **Visual Studio**
3. Run the program

## How It Works
Players take turns on a 3x3 grid, and the first to align three marks (horizontally, vertically, or diagonally) wins.

## Technologies Used
- **C#**
- **Windows Forms (GUI)**
